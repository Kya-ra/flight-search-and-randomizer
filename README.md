## csu33012-2526-project22

# SkySaver - The Budget Flight Finder

# Instructions for using Docker (local users only)

# 1. Clone repo
git clone <https://gitlab.scss.tcd.ie/csu33012-2526-group22/csu33012-2526-project22.git>

cd csu33012-2526-project22

# 2. Set API key
export SERPAPI_API_KEY=your_key_here

Get your key from: https://serpapi.com/manage-api-key

If this command fails, create a copy of .env.example and call it .env
Make sure this file is in your .gitignore as you don't want to push an API key
Put the API key in your .env

# 3. Launch container
docker compose up

# 4. Enjoy
The frontend will not be running on https://localhost:5173
For certain network setups, you may need to use the IP generated by Vite, which will be in the docker output as follows:

frontend | VITE v 7.2.2 ready in 200ms
frontend |
frontend | -> Local:        https://localhost:5173
frontend | -> Network:  <IP ADDRESS HERE>

# Instructions for not using docker (coder users)
These instructions assume you have the default SCSS coder MySQL database already running. For instructions on setting this up, please refer to the Labsheet.

# 1. Clone repo
git clone <https://gitlab.scss.tcd.ie/csu33012-2526-group22/csu33012-2526-project22.git>

cd csu33012-2526-project22

# 2. Set API key
export SERPAPI_API_KEY=your_key_here

Get your key from: https://serpapi.com/manage-api-key

If this command fails, create a copy of .env.example and call it .env
Make sure this file is in your .gitignore as you don't want to push an API key
Put the API key in your .env

# 3. Start backend

mvn clean install -f backend/pom.xml -DskipTests
mvn spring-boot:run -f backend/pom.xml

# 4. Start frontend

cd frontend
npm install && npm run dev -- --host

# 5. Enjoy
The frontend will not be running on https://localhost:5173
For certain network setups, you may need to use the IP generated by Vite, which will be in the shell output as follows:

VITE v 7.2.2 ready in 200ms

Local:        https://localhost:5173
Network:  <IP ADDRESS HERE>
